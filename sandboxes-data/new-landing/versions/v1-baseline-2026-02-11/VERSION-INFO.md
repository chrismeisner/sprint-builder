# Version 1 - Baseline (Feb 11, 2026)

## Overview
This is the initial baseline version of the Chris Meisner portfolio landing page. This version represents a clean, production-ready state that can serve as a reference point for future iterations.

## What's in this version

### Design & Layout
- Single-page portfolio with fixed sidebar navigation (desktop only)
- Responsive design that adapts to mobile
- Clean, minimalist aesthetic with strong typography
- Dark mode support (system preference + keyboard shortcut 'd')

### Content Sections
1. **About** - Profile photo + bio introducing Chris as product/brand designer
2. **Capabilities** - 4 capability areas: Brand Foundation, Visual System, Product Experience, Go-to-Market
3. **Projects** - 3 placeholder projects with slideshow functionality
4. **Services** - Detailed service offerings:
   - Brand Sprint (3 weeks, $15k)
   - Product Sprint (4 weeks, $25k)
   - Fractional Designer (20hrs/week, $5k/week)
5. **Contact** - Email + social links

### Technical Details
- Built with: HTML, Tailwind CSS, Vite
- External fonts: Inter (body), UnifrakturCook (decorative)
- Dark mode implementation: CSS classes + system preference detection
- Interactive features: Smooth scroll, project slideshow click-to-advance

### Known Placeholder Content
- Project images (01a.png, 01b.png, 02a.png, 03a.png)
- Project names and descriptions are generic
- Social media links point to base URLs

## Use Cases for This Version
- Reference for original content structure
- Rollback point if future iterations go off track
- Comparison baseline for A/B testing new layouts
- Template for creating similar pages
